1. בתיקיית גיט יש מעקב על השינויים בקוד

3. .git היא עוקבת על השינויים
4. לא
5. גם בתיקייה שיש קבצים

## תרגיל 2

1. git init -> git add -> git commit.
2. git add about.html -> git commit -m "added about file"
3. קובץ שנמצא בבמה מוכן כבר להישמר בשמירת גירסא הבאה
